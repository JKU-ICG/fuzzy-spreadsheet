To explore the features of Fuzzy Spreadsheet please follow the instructions. These set of instructions will guide you to install and explore the features in a demo sheet which has been provided for you.

Pre-Installation Checklist

Any of the latest browsers such as Firefox, Chrome, Edge.

Installation

  1. Download manifest.xml file in your Downloads folder.
  2. Open the Excel Online document.
  3. Go to Insert → Office Add-Ins → Manage My Add-Ins → Upload My Add-In (as shown in the figure)
  4. Upload locally stored manifest.xml file.
  5. Open Fuzzy Spreadsheet side panel by clicking on ‘Show Taskpane’ (Home → Commands Group → Show taskpane)
